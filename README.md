# PixelLogic
A deeper look into Notch's toy "circuits" project

# Build instructions
- `git clone --recursive PixelLogic`
- `cd libs/glbinding && mkdir build`
- `cd build`
- `cmake ..`
- `make`
- `cd .. && cd ..`
- `mkdir build && cd build`
- `cmake ..`
- Build with `make` or `visual studio` if on windows.
